利用 for 迴圈執行變數增量或重複執行特定程序.

Python for 迴圈

Python

for i in range(1, 10, 1): 
    print(i)

有關變數設定, 起始值使用, for 迴圈應用, 函式定義與呼叫

"""利用 def 定義一個函式, 名稱為 addto, 輸入變數為 upto
進入函式後定義一個變數名稱為 sum 起始值為 0
執行重複迴圈時, 從 1 開始逐步增量 1, 直到 i 的值小於 upto+1
期間 sum 累加上 i 的值, 每次執行迴圈都會印出 sum 的值
"""
def addto(upto):
  sum = 0
  for i in range(1, upto+1, 1):
    print("現在 sum 的值為: " + str(sum))
    sum = sum + i
    print("加上 " + str(i) + "之後, sum 的值為: " + str(sum))
    print("="*30)
addto(10)
